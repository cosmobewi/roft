Work Packages
=============

Introduction
------------
Les Work Packages (WP) structurent le développement de la **ROFT — Rhythm of Fields Theory** :
d’abord les bases théoriques (RG en langage rythme, champ d’horloges, pont avec la MQ),
puis l’implémentation & les tests, les extensions dynamiques,
et enfin les applications cosmologiques.
